Resource volterra_origin_pool
=============================

The Origin Pool allows CRUD of Origin Pool resource on Volterra SaaS

~> **Note:** Please refer to [Origin Pool API docs](https://docs.cloud.f5.com/docs/api/views-origin-pool) to learn more

Example Usage
-------------

```hcl
resource "volterra_origin_pool" "example" {
  name                   = "acmecorp-web"
  namespace              = "staging"
  endpoint_selection     = ["endpoint_selection"]
  loadbalancer_algorithm = ["loadbalancer_algorithm"]

  origin_servers {
    // One of the arguments from this list "vn_private_ip vn_private_name segment_name public_name k8s_service consul_service custom_endpoint_object public_ip private_ip private_name segment_ip" must be set

    public_name {
      dns_name         = "value"
      refresh_interval = "20"
    }

    labels = {
      "key1" = "value1"
    }
  }

  // One of the arguments from this list "port automatic_port lb_port" must be set
  lb_port = true

  // One of the arguments from this list "use_tls no_tls" must be set
  no_tls = true
}

```
